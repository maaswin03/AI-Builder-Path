# Prompt Security & Caching Refactor

## Overview

This repository contains the solution for the **Prompt Security & Caching Refactor** assignment. The objective is to redesign an AI-powered HR assistant prompt by improving prompt caching efficiency and strengthening security against prompt injection attacks.

---

## Objectives

* Identify static and dynamic sections of the original prompt.
* Improve prompt caching efficiency.
* Remove sensitive information from the prompt.
* Design a secure prompt architecture.
* Define mitigation strategies against prompt injection attacks.

## Conclusion

The redesigned prompt separates reusable instructions from dynamic user data, improving caching efficiency while protecting confidential employee information from prompt injection attacks.
